# DETE-Exit-Survey-Analysis-using-SQL-Power-BI-
This project analyzes employee exit survey data from the Department of Education, Training and Employment (DETE) to identify key factors influencing employee resignations. The goal is to uncover trends related to dissatisfaction, tenure, age group, and reasons for leaving, and to present insights through interactive dashboards.

-- Objectives
1. Clean and transform raw exit survey data using SQL,
2. Analyze resignation reasons and demographic patterns,
3. Identify dissatisfaction trends across employee groups,
4. Build an interactive Power BI dashboard for data-driven insights.

-- Tools & Technologies

1. SQL – Data cleaning, filtering, joins, aggregations,
2. Power BI – Data modeling, DAX measures, interactive visualizations,
3. Excel / CSV – Source data handling.
