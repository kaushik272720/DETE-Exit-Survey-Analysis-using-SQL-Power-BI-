# DETE-Exit-Survey-Analysis-using-SQL-Power-BI-
This project analyzes employee exit survey data from the Department of Education, Training and Employment (DETE) to identify key factors influencing employee resignations. The goal is to uncover trends related to dissatisfaction, tenure, age group, and reasons for leaving, and to present insights through interactive dashboards.

Objectives
Clean and transform raw exit survey data using SQL
Analyze resignation reasons and demographic patterns
Identify dissatisfaction trends across employee groups
Build an interactive Power BI dashboard for data-driven insights

Tools & Technologies

SQL – Data cleaning, filtering, joins, aggregations
Power BI – Data modeling, DAX measures, interactive visualizations
Excel / CSV – Source data handling
